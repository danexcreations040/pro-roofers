Update your admin login password. Keep it safe — only you should know it.

🔑 How to Change Your Password

Your password is the ADMIN_TOKEN you set in Netlify. To change it:

  1. Go to netlify.com → your site → Site configuration
  2. Click Environment variables
  3. Find ADMIN_TOKEN and click Edit
  4. Type your new password → Save
  5. Click Deploys → Trigger deploy
